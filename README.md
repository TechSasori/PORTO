# PORT

To test the code, it is necessary to create and populate a localhost database provided by PostgreSQL using the text files.
To insert the ship images, they must be located in a public directory on your computer, such as 'C:/', and you must choose a JPG file, for example: 'C:/Navio1.jpg'.
After that, write the database access password in the connection function in the Python code persistencia.py and install the required libraries. Then simply compile and run the code.
